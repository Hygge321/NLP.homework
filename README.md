# LSTM

本项目在不使用Pytorch包装好的LSTM函数下，实现了在单层LSTM与双层LSTM。

## 文件介绍

LSTM.py文件为使用Pytorch内包装好的LSTM函数

LSTMLM.py文件为手动实现**单层LSTM**

doubleLSTMLM.py文件为手动实现**双层LSTM**
